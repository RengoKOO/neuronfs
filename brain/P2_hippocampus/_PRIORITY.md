# P2: HIPPOCAMPUS ??PRIORITY 2

## Subsumption Rule
- Memory provides past cases to P3~P6 decisions
- Permission: AUTO-ACCUMULATE (agent records)
- Time Scale: EVENT-DRIVEN
- Analogy: Diary. Records experience but does not judge

## File Types
- .memory = episodic record (what succeeded/failed when)
- rewards/ = dopamine history
- failures/ = bomb history

## Forgetting Mechanism
- Counter Decay: old .memory files lose counter value
- Pruning: counter=0 .memory files are deleted
